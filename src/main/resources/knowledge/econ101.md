# 微观经济学

## 课程概述
微观经济学研究个体经济单位（家庭、企业）的经济行为，探讨资源配置与价格决定机制。本课程是经管类专业的核心基础课。

## 第一章：供需理论

### 需求定律
在其他条件不变的情况下，商品价格上升，需求量下降；价格下降，需求量上升。影响因素包括：
- 商品自身价格（沿需求曲线移动）
- 消费者收入（需求曲线平移）
- 替代品价格
- 互补品价格
- 消费者偏好和预期

### 供给定律
在其他条件不变的情况下，商品价格上升，供给量增加；价格下降，供给量减少。影响因素：
- 生产技术
- 生产要素价格（成本）
- 生产者数量
- 政府政策（税收、补贴）

### 市场均衡
当需求量等于供给量时的价格称为均衡价格。市场机制会自动调整使价格趋近于均衡：
- 价格高于均衡 → 供过于求 → 价格下降
- 价格低于均衡 → 供不应求 → 价格上升

## 第二章：消费者行为

### 边际效用递减
随着消费者对某种商品消费量的增加，每增加一单位消费所带来的额外满足感（边际效用）是递减的。例如：吃第一个汉堡很满足，吃第五个就腻了。

### 无差异曲线
表示给消费者相同满足程度的不同商品组合。特点：
- 离原点越远，效用越高
- 任意两条无差异曲线不相交
- 凸向原点（边际替代率递减）

### 消费者均衡
消费者在预算约束下追求效用最大化。均衡条件：各商品的边际效用与价格之比相等。

## 第三章：生产成本

### 短期与长期
- **短期**：至少有一种生产要素（通常是资本）固定不变
- **长期**：所有生产要素都可以调整

### 边际成本与平均成本
- 边际成本曲线呈 U 型，先降后升（边际报酬递减规律）
- 边际成本曲线与平均成本曲线相交于平均成本的最低点

### 规模经济
长期平均成本随产量增加而下降的现象。来源：专业化分工、技术优势、采购议价能力。但规模过大可能导致规模不经济（管理效率下降）。

## 第四章：市场结构

### 完全竞争
- 特征：大量买者卖者、产品同质、自由进出、信息完全
- 企业是价格接受者
- 长期均衡：价格 = 边际成本 = 最低平均成本

### 垄断
- 特征：单一卖者、无替代品、进入壁垒
- 垄断者定价高于边际成本，造成社会福利损失
- 形成原因：自然垄断、专利、政府授权、资源控制

### 寡头垄断与垄断竞争
- 寡头：少数几家大企业主导市场，互相依存（博弈论分析）
- 垄断竞争：许多企业、产品有差异、自由进出（如餐饮业）
